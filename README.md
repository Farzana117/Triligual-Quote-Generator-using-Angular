**Trilingual Quote Generator (Angular 17)**
A modern, multilingual Quote Generator App built using Angular 17 (Standalone Components).
This application displays motivational, success, mindfulness, and study quotes in English, Hindi, and Telugu and allows users to filter categories, save favourites, and receive toast notifications.

**Random Quote Generator**
 -> Displays a random quote based on selected category
 -> Quotes translated into English, Hindi, and Telugu

**Multilingual Support**
-> Real-time language switching (ENGLISH / HINDI / TELUGU)
-> Language stored using localStorage
-> No page reload required

**Tech Stack**
Frontend
 -> Angular 17
 -> TypeScript
 -> HTML
 -> CSS

**Angular Concepts Used**
 -> Standalone Components
 -> Signals & Computed State
 -> Dependency Injection
 -> Custom Services
 -> LocalStorage persistence
 -> Toast notification component
